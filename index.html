FormShell v1.0

Interactive multi-step forms in the browser console

Welcome!

FormShell is a framework for creating interactive multi-step forms directly in the browser console, offering an elegant and modern user experience.

Built with TypeScript

IMPORTANT: Open the Browser Console!

To use the demo, you need to open the browser's JavaScript console:

• Chrome/Edge: Press F12 or Ctrl+Shift+J (Cmd+Option+J on Mac)

• Firefox: Press F12 or Ctrl+Shift+K (Cmd+Option+K on Mac)

• Safari: Enable Developer menu, then press Cmd+Option+C

Getting Started

  1. Open the Browser Console (see above)
  2. Choose an example below by clicking the card
  3. In the console, type formShell.start() to begin
  4. Use formShell.answer("...") to respond to questions
  5. Type formShell.help() at any time to see all commands
  6. Type formShell.continue() to resume after viewing help

Main Commands

  • formShell.start() Start the form
  • formShell.help() Show all commands (non-blocking)
  • formShell.continue() Resume after viewing help
  • formShell.answer("...") Answer and proceed
  • formShell.skip() Skip question (if optional)
  • formShell.y() / formShell.n() Answer Yes/No (shortcut)
  • formShell.back() Go to previous question
  • formShell.submit() Submit data (at the end)
  • formShell.reset() Start over
